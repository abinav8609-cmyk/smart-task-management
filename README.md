# Smart Task Management System

A simple Python CLI application to manage your daily tasks.

## Features
1. **Add Task** - Add new tasks to your list
2. **View Tasks** - Display all your current tasks  
3. **Mark Task as Complete** - Remove completed tasks from the list
4. **Exit** - Close the application

## How to Run
1. Make sure you have Python installed
2. Run the program:
```bash
python main.py
